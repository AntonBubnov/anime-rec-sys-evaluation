# Anime Recommendation System Evaluation

This project is dedicated to exploring, implementing, and benchmarking different recommendation system architectures using the **Anime Recommendation Database 2020**.

## 🎯 Project Goals
- **Data Preprocessing:** Cleaning and merging large-scale datasets (anime, ratings, and synopses).
- **Exploratory Data Analysis (EDA):** Understanding user preferences and anime trends.
- **Model Development:** Implementing multiple approaches (Popularity-based, Content-based, Collaborative Filtering).
- **Comparative Evaluation:** Measuring and comparing model performance in a unified environment.

## 📂 Project Structure
- `data/`: Raw and processed datasets (excluded from Git).
- `notebooks/`: 
  - `01_eda_preprocessing.ipynb`: Data cleaning and merging.
  - `02_model_...ipynb`: Specific recommendation models.
  - `04_evaluation.ipynb`: Final comparison and metrics.
- `src/`: Utility scripts and shared functions.

## 🛠 Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-learn
- **Tools:** VS Code, Jupyter Notebooks, Git

## 📊 Dataset
The project uses the [Anime Recommendation Database 2020](https://www.kaggle.com/datasets/hernan4444/anime-recommendation-database-2020) from Kaggle.
